# Graph-RAG-Based Structured Text Reader

> A lightweight plug-in built on Microsoft’s GraphRAG for extracting strongly-structured text from PowerPoint files.

## What it does

- Slices the PPT, runs OCR on each slide, and converts the output to clean Markdown.
- Applies a **proprietary readability algorithm** (script not published) that dramatically improves the legibility of OCR results.

## Benchmarks

Across multiple evaluations, the plug-in consistently boosts large-model performance on structured-document tasks.

# 基于 Graph RAG 的结构性文本读取工具

> 这是基于微软 GraphRAG 的小插件，用于读取 PPT 等强结构性文本。

## 功能

- 运用 PPT 切片进行 OCR 处理，转化成 Markdown 格式。
- 根据开发的一种**特殊算法**（脚本未上传），让 PPT 的 OCR 结果可读性更强。

## 性能表现

在多次测试中，该插件都能让大模型在处理强结构性文本方面表现出更优秀的性能。
